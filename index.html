What is GeoAlert?

GeoAlert pulls live data from NASA's Earth Observatory Natural Event Tracker (EONET v3) — the same data NASA scientists use — and presents it in a way anyone can understand and act on.

🗺️

Map View

Interactive world map with color-coded markers for every event type. Click any marker to see full details.

📋

Events List

Browse all events as cards. Sort by date, name or category. Click a card for full details.

⏱️

Timeline

See daily counts, category breakdown and weekday activity charts to spot patterns over time.

📍

Near Me

Type your city or share your GPS location to see events within a chosen radius — with distance and safety context.

🎓

Learn

Deep-dive into each event type: what causes it, how dangerous it can be, and how to stay safe.

📊

Statistics

KPI cards, pie charts, trend lines and a ranked top-10 events table — all from your current filter set.

Pro tip: Use the left sidebar filters to narrow by category, date range or keyword — then click Apply Filters to update every view at once.

🗺️ Using the Map

1
Navigate — Scroll to zoom, click and drag to pan. Use the +/− buttons bottom-right.
2
Click a marker — A popup shows the event name, category badge, date and magnitude. Hit View Details → to open the full detail panel.
3
Clustered circles — When many events are close together they cluster. Click a cluster number to zoom in and split them apart.
4
Map Layer — In the left sidebar choose Standard, Satellite, Topographic, or Dark map.
5
Top-right controls — ⛶ Fit All zooms to show every event. ⊞ Cluster toggles grouping. 🔥 Heatmap (coming soon).
6
Legend — Bottom-left shows the color/icon key for event categories currently on the map.

🔍 Filters & Search

All filters live in the left sidebar. Change them then hit Apply Filters to reload data from NASA.

Filter
What it does
Keyword Search
Instantly filters visible events by title or description — no Apply needed, updates as you type.
Status
Open = still active. Closed = resolved. All shows both.
Time Window
Pick 3 / 7 / 14 / 30 days, or choose Custom to set exact start and end dates.
Max Events
Cap how many events are loaded (20–200). Lower = faster; higher = more complete.
Categories
Check/uncheck individual event types. The number badge shows how many events are in each category right now.
Map Layer
Change the background map style: Standard, Satellite, Topo, or Dark.
Clear All
Resets every filter to defaults in one click.
Tip: Keyword search works live — start typing "fire" or "storm" and the event list updates immediately without pressing Apply.

📍 Near Me — Find Events Around You

1
Click "Use My Location" — your browser will ask permission for GPS. Allow it and GeoAlert will find your coordinates automatically.
2
Or type a location — Enter a city name like Tokyo, a country like Brazil, or raw coordinates like 51.5, -0.1 (lat, lon).
3
Set the radius — Choose 100 km, 200 km, 500 km or 1000 km to control how wide to search.
4
The mini-map shows your location (blue dot), radius circle, and nearby event markers. The list on the right is sorted closest-first.
5
Each result shows a safety note — yellow for nearby, orange for very close — with advice to check local authorities.
6
Click any event card or map marker to open the full detail panel.
Important: GeoAlert provides awareness only. Always follow instructions from your local emergency services for immediate safety.

📊 Other Views Explained

Three charts at the top: a daily bar chart (how many events per day), a category doughnut (share of each type), and a weekday bar (which days have most activity). Below is a scrollable chronological list of all events — click any to open details.

Five KPI cards at the top: total events, active count, closed count, categories count, and regions. Then a horizontal category bar chart, a status pie chart, and a 30-day trend line. At the bottom is a ranked Top 10 Events table — click any row to see full details.

Click any category card (Wildfires, Floods, etc.) to open a detailed knowledge modal with: a plain-English explanation, key facts table, safety tips, live EONET examples from current data, and links to authoritative external resources (USGS, FEMA, NHC etc.).

Slides in from the right whenever you click any event. Contains: a mini-map zoomed to the event location, geometry timeline (historical coordinate points over time), source links to NASA and curator sites, and the event's unique ID. Close with ✕ or press Esc.

Click the icon in the top bar. A summary card shows your current filter state, event count, and top 3 events. Copy the URL (restores your exact filters) or copy plain text to paste anywhere.

❓ Frequently Asked Questions

All event data comes from NASA's Earth Observatory Natural Event Tracker (EONET) v3 API. EONET aggregates data from USGS, NOAA, InciWeb, and other authoritative curators.

Events are updated by NASA's curators continuously. GeoAlert caches each API response for 3 minutes — so hitting refresh more than once per 3 minutes won't re-fetch. Click 🔄 Refresh in the top bar to manually clear the cache.

NASA's EONET API occasionally goes into maintenance or has high traffic causing 503 errors. When that happens, GeoAlert automatically shows a representative set of real historical events so the app stays functional. It silently retries every 60 seconds and will switch back to live data automatically.

Yes. GeoAlert is mobile-first and works on any modern smartphone browser. The sidebar collapses on small screens — use the top navigation buttons instead. The Near Me GPS feature works on mobile browsers that support geolocation.

Yes — use the Near Me view and type a country or region name, then increase the radius to 500–1000 km. The bounding box search will pull only events from that area. The main map also lets you visually identify regional clusters.

Open = the event is still active and being tracked by NASA's curators. Closed = the event has ended and a closing date has been recorded. By default, GeoAlert shows only Open events — change the Status filter to see Closed or All.

For storms: wind speed in knots. For temperature extremes: degrees Celsius. For fires/floods it's often not available (—). The magnitude value and unit come directly from NASA's EONET data for each individual event.

Click the Share button (top bar). Copy the URL — it encodes your current filters so anyone who opens it sees exactly what you see. You can also copy the plain-text summary to paste into messaging apps or social media.
